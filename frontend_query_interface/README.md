<!-- GETTING STARTED -->

### Prerequisites

Ensure you have Node.js installed

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/dyte-submissions/november-2023-hiring-AnshAtreja.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Download Elastic Search on your system, follow https://www.elastic.co/guide/en/elasticsearch/reference/current/zip-windows.html
4. Once downloaded, open cmd in the folder and run ( this may take some time )
  ```
  .\bin\elasticsearch.bat
  ```
5. Once the service is started, go inside the config folder, go to certs folder and find a file named as http_ca.crt.
6. Copy this file to your backend_log_ingestor folder.
7. Now run command in the backend_log_ingestor terminal to start server
```
  node server.js
```
7. Now you can ingest the data on http://localhost:3000/logs with json content as body.
8. Now go to your frontend_query_interface folder terminal and write npm run start to start the react app
9. You may now use the app
